## 🎯 每个文件夹包含内容

每个公式文件夹包含：
- `README.md` - 公式详细说明
- `visualization.html` - 最先进的HTML5可视化
- `theory.md` - 理论背景
- `examples/` - 应用示例
- `assets/` - 资源文件

## 🚀 技术栈

- **前端**: HTML5 + CSS3 + JavaScript ES6+
- **3D渲染**: Three.js + WebGL 2.0
- **数学渲染**: MathJax 3.0
- **动画**: GSAP + CSS Animations
- **交互**: 响应式设计 + 触摸支持
- **性能**: Web Workers + GPU加速
