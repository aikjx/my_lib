#### **批量重命名脚本**
```python
import os
import shutil

def standardize_filename(old_name, category, topic, version="v1.0"):
    """标准化文件名"""
    ext = os.path.splitext(old_name)[1]
    new_name = f"{category}-{topic}-{version}{ext}"
    return new_name

# 重命名映射表
rename_map = {
```
